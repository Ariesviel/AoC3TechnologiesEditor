# AoC3TechnologiesEditor

EN
- A tool for editing the technology tree in **Age of History 3**.
- UI created with `PySide6` — runs on Windows.
- Includes a script for reading `.json` (pseudo-JSON) files from the game directory.

RU
- Инструмент для редактирования технологического дерева в **Age of History 3**.
- Интерфейс создан с `PySide6`. Запускается на Windows.
- Реализован скрипт для чтения `.json` (псевдо-JSON) файлов из папки игры.
